## 📌 ¿Qué es esto?

Este repositorio es mi espacio de práctica en C++ para la [UTN-FRBA](https://www.frba.utn.edu.ar/).

Guardo ejercicios resueltos que me van dejando en diferentes materias, programas de práctica y plantillas de pseudocódigo.

Mi objetivo es **documentar el proceso de aprendizaje**, con todos sus errores y avances.
